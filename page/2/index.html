mongodb grammer

By jxc

Published 2015-10-27

mongodb 的环境是ubuntu 14.04，下载好ubuntu 后，直接运行mongod -f configfilepath


客户端端口默认是27017, 可以用db.getMongo

mongodb 支持的结构是BSON，2进制存储，据说是比JSON多出了date类型和类型


document => collection => database

命令和例子

show collections


db.collection.find()


db.collection.insert()


db.collection.update(query,update,{upsert:boolean,multi:boolean})


db.person.update({age: {$lt: 30}}, {$set: {status:”X”}}, {multi:true});


save命令可以更新或插入一个新文档，与update命令不同的是，save只能对一个文档进行操作。


大于($gt)、大于等于($gte)、小于($lt)、小于等于($lte)、不等于($ne)、包含于($in)、不包含于($nin)


db.persion.insert({name:”jxc”, age:11});


db.collection.remove(query,justOne)


query 是一个BSON结构

数组条件:


数组完全匹配：

db.inventory.find( { tags: [ ‘fruit’, ‘food’, ‘citrus’ ] } );
单个元素匹配：

db.inventory.find( { tags: ‘fruit’ } );
特定元素匹配:

db.inventory.find( { ‘tags.0’ : ‘fruit’ } );


子文档匹配，类似数组的特定元素匹配

复合查询


db.inventory.find({$and:[{type: “food”},{ price: { $lt: 95 }}]});


db.person.find({$or:[{status:”A”}, {age: 40}]});

游标


find命令并不直接返回结果，而是返回一个结果集的迭代器，即游标。

highlight ruby
/ 方法一 /
var myCursor = db.inventory.find( { type: “food” } );
var myDocument = myCursor.hasNext() ? myCursor.next() : null;

if (myDocument) {
var myItem = myDocument.item;
print(tojson(myItem));
}
/ 方法二 /
var myCursor = db.inventory.find( { type: “food” } );
myCursor.forEach(printjson);
endhighlight

数量限制


db.person.find().limit(3)

限制集


db.collection.find( 查询条件,子段条件 )


字段条件：JSON对象，格式=>{字段：值},值等于1时表示需要返回,0时表示不需要


返回person的”status”和”age”字段: db.person.find({}, {status:1,age:1})
